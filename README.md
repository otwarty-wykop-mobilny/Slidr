## Purpose

fork of `Slidr` optimized for [WykopMobilny](https://github.com/otwarty-wykop-mobilny/wykop-android). Works on Google Pixel landscape mode

## Usage
```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("com.github.usefulness:slidr:$slidrVersion")
}
```
